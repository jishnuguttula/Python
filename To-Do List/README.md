# To-Do List Application (Python)

This project is a simple command-line To-Do List application built using Python to manage daily tasks.

## 🚀 Features
- Add new tasks
- View all tasks
- Remove tasks
- Simple menu-driven program

## 🛠️ Technologies Used
- Python
- Lists
- Loops and condition statements

## ▶️ How to Run
1. Open terminal or command prompt
2. Run the file:
   python todo_list.py
3. Choose options from menu:
   1 - Add Task
   2 - View Tasks
   3 - Remove Task
   4 - Exit

## 📌 Example
Input:
1
Enter task: Study Python

Output:
Task added successfully

## 🎯 Purpose
This project demonstrates the use of lists, loops, and user input handling in Python.
