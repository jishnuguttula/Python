# Contact Book (Python)

This is a simple Python-based Contact Book application that allows users to manage contacts efficiently.

## 🚀 Features
- Add new contacts
- Display all contacts
- Search for a contact
- Delete a contact
- Prevent duplicate entries

## 🛠️ Technologies Used
- Python
- Dictionary
- Functions and Loops

## ▶️ How to Run
1. Run the Python file:
   python contact_book.py
2. Select options from the menu

## 📌 Example
Add → View → Search → Delete contacts easily

## 🎯 Purpose
This project demonstrates the use of dictionaries, functions, and control statements in Python.
