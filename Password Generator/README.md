# Password Generator (Python)

This project is a simple Python-based password generator that creates strong and random passwords based on user-defined length.

## 🚀 Features
- Generates random secure passwords
- Supports custom password length
- Uses letters, numbers, and special characters

## 🛠️ Technologies Used
- Python
- random module
- string module

## ▶️ How to Run
1. Open terminal or command prompt
2. Run the file:
   python password_generator.py
3. Enter desired password length
4. Get your generated password

## 📌 Example
Input:
Enter length: 8

Output:
A@7kL#2p

## 🎯 Purpose
This project helps in understanding randomization, string handling, and basic Python modules.
